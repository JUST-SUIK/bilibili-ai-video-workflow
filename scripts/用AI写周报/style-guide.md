# 视觉风格指南：用AI 10分钟写完一周周报

## 视觉调性

**选择：暗黑科技风（Dark Tech）**

**理由：**
- AI/效率工具主题天然适合深色科技风，B站知识区同类视频中深色风格辨识度更高
- 深色背景让信息层次更清晰，文字和动画元素更突出
- 蓝橙双色对比在深色背景上视觉冲击力强，符合"打工人效率工具"的利落感
- 整体基调：**专业、科技、利落、现代**

**视觉关键词：**
- 深色背景（#0a0a0f ~ #0d0d14）
- 蓝橙双强调色
- 网格/几何线条装饰
- 微妙光晕和脉冲效果
- 干净的卡片布局

## 色彩体系

| 角色 | 色值 | 用途 |
|------|------|------|
| 主色 | `#4361ee` | 标题、强调、CTA按钮、AI相关元素 |
| 辅色 | `#7c5cff` | 次要元素、渐变搭配、光晕 |
| 背景色 | `#0a0a0f` | 页面主背景 |
| 文字色 | `#f5f5f7` | 正文文字、标题 |
| 强调色 | `#ff5a2c` | 数据高亮、重点标注、周报相关 |
| 表面色 | `rgba(255,255,255,0.06)` | 卡片、容器背景 |
| 弱文字色 | `rgba(255,255,255,0.5)` | 注释、说明文字 |
| 渐变起始 | `#4361ee` | 科技感渐变起点 |
| 渐变结束 | `#7c5cff` | 科技感渐变终点 |

**配色说明：**
- 主色采用电光蓝（#4361ee），传达专业、科技、效率
- 辅色采用霓虹紫（#7c5cff），与主色形成蓝紫渐变，体现AI感
- 强调色采用番橙（#ff5a2c），用于周报相关内容，增加温暖感和视觉冲击
- 背景采用近黑（#0a0a0f），比纯黑#000000更柔和

**渐变组合：**
- 主渐变：`linear-gradient(135deg, #4361ee, #7c5cff)` — 用于AI元素、标题装饰
- 弱渐变：`linear-gradient(135deg, rgba(67,97,238,0.15), rgba(124,92,255,0.15))` — 用于卡片背景
- 强调渐变：`linear-gradient(135deg, #ff5a2c, #ff8c32)` — 用于周报高亮

## 字体体系

| 层级 | 字体 | 字号 | 字重 | 行高 | 用途 |
|------|------|------|------|------|------|
| H1 | Noto Sans SC / Inter | 72-96px | Bold (700) | ≥1.5 | 封面标题 |
| H2 | Noto Sans SC / Inter | 48-64px | Bold (700) | ≥1.5 | 段落标题 |
| H3 | Noto Sans SC / Inter | 32-40px | SemiBold (600) | ≥1.5 | 小标题 |
| Body | Noto Sans SC / Inter | 24-32px | Regular (400) | ≥1.6 | 正文 |
| Caption | Noto Sans SC / Inter | 18-24px | Regular (400) | ≥1.5 | 注释、元数据 |
| Code | JetBrains Mono / Fira Code | 20-28px | Regular (400) | ≥1.6 | 代码、Prompt模板 |

**⚠️ 字体规范（强制执行）：**
- **禁止使用 Inter 字体** — 在 Playwright 录制时会出现字重抖动，改用 Geist 或 Noto Sans SC
- **禁止**对 `font-size` 做CSS动画（会导致字体抽动）
- **禁止**对 `letter-spacing` 做CSS动画
- **禁止** `blur → 清晰` 的过渡效果（直接用 `opacity` 淡入或直接显示）
- 文字容器**必须**设置固定 `min-height`
- 行高**必须** ≥ 1.5，防止上下行文字覆盖
- 所有文字**必须** `-webkit-font-smoothing: antialiased`
- 文字进场后位置固定，不做任何位移/缩放微调
- 中文正文使用 `font-family: 'Noto Sans SC', 'Geist', sans-serif`
- 代码块使用 `font-family: 'JetBrains Mono', monospace`
- **禁止**引入未定义的字体（如Archivo Black、Caveat等）

## 动画风格

| 维度 | 选择 | 说明 |
|------|------|------|
| 缓动曲线 | `cubic-bezier(0.25, 0.46, 0.45, 0.94)` | 统一使用，禁止linear |
| 进场方式 | 淡入 + 上移 | 全片统一，`opacity: 0→1, translateY: 20px→0` |
| 运动节奏 | 快 0.3s / 中 0.5s / 慢 0.8s | 根据元素重要性调整 |
| 循环动画 | 脉冲 + 漂浮 + 渐变流动 | 持续视觉活力 |
| 转场方式 | 交叉溶解 | 全片统一，禁止瞬间切换 |

**动画参数详解：**

**进场动画（统一）：**
```css
@keyframes fadeInUp {
  from {
    opacity: 0;
    transform: translateY(20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

/* 应用 */
.element {
  animation: fadeInUp 0.5s cubic-bezier(0.25, 0.46, 0.45, 0.94) forwards;
}
```

**脉冲动画（用于AI图标/强调元素）：**
```css
@keyframes pulse {
  0%, 100% {
    transform: scale(1);
    opacity: 1;
  }
  50% {
    transform: scale(1.05);
    opacity: 0.8;
  }
}

.ai-icon {
  animation: pulse 2s cubic-bezier(0.25, 0.46, 0.45, 0.94) infinite;
}
```

**漂浮动画（用于装饰元素）：**
```css
@keyframes float {
  0%, 100% {
    transform: translateY(0px);
  }
  50% {
    transform: translateY(-10px);
  }
}

.decoration {
  animation: float 3s cubic-bezier(0.25, 0.46, 0.45, 0.94) infinite;
}
```

**渐变流动（用于背景装饰）：**
```css
@keyframes gradientShift {
  0% { background-position: 0% 50%; }
  50% { background-position: 100% 50%; }
  100% { background-position: 0% 50%; }
}

.gradient-bg {
  background: linear-gradient(135deg, #4361ee, #7c5cff);
  background-size: 200% 200%;
  animation: gradientShift 4s cubic-bezier(0.25, 0.46, 0.45, 0.94) infinite;
}
```

**动画时长规范：**
- 快速（0.3s）：按钮反馈、图标切换、微交互
- 中速（0.5s）：卡片进场、文字淡入、列表项
- 慢速（0.8s）：页面转场、标题进场、重要元素

## 转场规范

**统一转场方式：交叉溶解（Cross Dissolve）**

| 转场类型 | 时长 | 缓动 | 适用场景 |
|----------|------|------|----------|
| 场景切换 | 0.8s | cubic-bezier | 段落之间、页面切换 |
| 元素进场 | 0.5s | cubic-bezier | 卡片、列表、图标 |
| 元素退场 | 0.3s | cubic-bezier | 消失的元素 |
| 强调出现 | 0.6s | cubic-bezier | 重点内容、CTA |

**禁止的转场方式：**
- ❌ 瞬间切换（无过渡）
- ❌ 硬切（直接跳转）
- ❌ 过于花哨的3D翻转
- ❌ 旋转过渡

## 禁止事项

**CSS动画禁止：**
- ❌ `font-size` 动画
- ❌ `letter-spacing` 动画
- ❌ `blur → 清晰` 过渡
- ❌ `linear` 缓动
- ❌ 纯黑 `#000000` 作为背景

**正确实现：**
```css
/* ✅ 正确：只动opacity和transform */
.element {
  animation: fadeInUp 0.5s cubic-bezier(0.25, 0.46, 0.45, 0.94) forwards;
}
```

**其他禁止事项：**
- 禁止引入未在本指南中定义的字体
- 禁止文字颜色与背景对比度低于4.5:1
- 禁止使用闪烁动画（可能引发光敏性癫痫）
- **禁止字号小于 18px**（在 1920×1080 屏幕上小于 18px 几乎不可读）

## 动画同步规则（强制执行）

**问题**：文字动画在页面加载时立即播放，但音频还没开始念，导致文字比旁白先出现。

**规则**：
- 引导语（如"一个反直觉的事实"）：0.5s 延迟出现
- 主体文字：延迟 3-5s，等引导语念完再开始逐字出现
- 数字/强调元素：延迟 7-10s，等主体文字念完再出现
- 底部信息：延迟 10s+

**公式**：`动画延迟 = 前一段音频时长 + 0.5s 缓冲`

**禁止**：文字动画比音频先出现。宁可文字晚一点出现，也不能让观众看到文字但还没听到解释。

## 全片风格统一规则（强制执行）

**禁止出现白色/浅色页面**。所有页面必须使用暗色背景（#0a0a0f ~ #0d0d14）。

原因：从暗色突然切换到亮色，观众眼睛受不了，风格也不统一。

如果需要区分不同内容区域，使用暗色系的深浅变化（如 #0a0a0f vs #1a1a2a），而不是切换到白色/蓝色。
