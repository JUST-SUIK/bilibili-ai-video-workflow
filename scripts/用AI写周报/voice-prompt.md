# MiMo TTS 音色设计提示词

## 用于 mimo-v2.5-tts-voicedesign 模型

### 中文版（推荐）

```
用山东口音，语速快，像朋友聊天一样分享好消息，带着兴奋和真诚，不做作。
```

### 详细版（需要更精细控制时使用）

```
一位二十多岁的年轻男性，说带轻微山东口音的普通话，语速偏快，声音清亮有活力。语气像是一个好朋友发现了一个特别好的东西，迫不及待想分享给你，带着兴奋和真诚。不是在念稿子，是在聊天。偶尔会有一点得意的笑，但不做作。
```

### 使用方式

调用 MiMo API 时，将此提示词放在 `role: user` 的 `content` 中：

```python
messages=[
    {"role": "user", "content": "用山东口音，语速快，像朋友聊天一样分享好消息..."},
    {"role": "assistant", "content": "你要合成的文本内容"}
]
```

### 重要注意事项

1. **必须使用 `mimo-v2.5-tts-voicedesign` 模型**，不要用 voiceclone 或预置音色
2. **每段音频结尾添加 1.5 秒静音**，用于转场间隔，避免两段音频紧贴
3. **API Key 用环境变量**，禁止明文写在代码或命令行中
4. **语速要快**，知识分享视频的旁白不能太慢，观众会失去耐心
5. **要有个性**，不能像机器人念稿，要像朋友聊天

### 音色调优记录

| 版本 | 音色描述 | 效果 | 问题 |
|------|----------|------|------|
| v1 | voiceclone（参考音频） | 声纹不是本人，有侵权风险 | 不再使用 |
| v3 | voicedesign，标准普通话，语速中等 | 太无趣，像念稿 | 语速太慢 |
| v3.1 | voicedesign，标准普通话，语速偏快 | 有改善但还是太正式 | 缺少个性 |
| v4 | voicedesign，山东口音，语速快，朋友聊天感 | 有个性，有活力 | 当前版本 |
