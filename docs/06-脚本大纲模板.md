## 脚本写作检查清单

### 内容检查
- [ ] 主题明确，有清晰的价值主张
- [ ] 逻辑通顺，有清晰的结构
- [ ] 内容有深度，不是泛泛而谈
- [ ] 有具体案例或演示
- [ ] 时长控制合理

### 表达检查
- [ ] 语言口语化，像在聊天
- [ ] 没有长句和复杂句式
- [ ] 有适当的停顿和节奏
- [ ] 重点突出，有强调

### 技术检查
- [ ] 画面描述清晰可执行
- [ ] 素材需求明确
- [ ] 时间分配合理
- [ ] 与整体风格一致
