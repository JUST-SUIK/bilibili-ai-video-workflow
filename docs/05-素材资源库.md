### 1.4 动画效果库

#### 进场动画
```css
/* 淡入上升（最常用） */
@keyframes fadeUp {
  from { opacity: 0; transform: translateY(20px); }
  to { opacity: 1; transform: translateY(0); }
}

/* 缩放弹入 */
@keyframes scaleIn {
  from { opacity: 0; transform: scale(0.8); }
  to { opacity: 1; transform: scale(1); }
}

/* 从左滑入 */
@keyframes slideLeft {
  from { opacity: 0; transform: translateX(-30px); }
  to { opacity: 1; transform: translateX(0); }
}
```

#### 缓动曲线
```css
/* 标准缓动 */
--ease-out-expo: cubic-bezier(0.16, 1, 0.3, 1);

/* 弹性缓动 */
--ease-out-back: cubic-bezier(0.34, 1.56, 0.64, 1);
```

#### 数据可视化
```css
/* 条形图生长 */
@keyframes barGrow {
  from { transform: scaleY(0); }
  to { transform: scaleY(1); }
}

/* 数字计数器 */
@keyframes countUp {
  from { opacity: 0; transform: scale(0.5); }
  to { opacity: 1; transform: scale(1); }
}
```

## 四、素材收集检查清单

每次制作视频前，按此清单收集素材：

### 视觉参考
- [ ] 确定视觉风格方向（暗黑科技/极简/故障风等）
- [ ] 确定配色方案（主色、辅色、强调色HEX码）
- [ ] 确定字体方案（标题、正文、代码）
- [ ] 收集3-5个设计参考链接
- [ ] 确认可用的HTML动画模板

### 技术资料
- [ ] 提取脚本中的技术声明（逐条列出）
- [ ] 验证每条技术声明的准确性
- [ ] 收集可引用的数据来源（标注URL）
- [ ] 整理关键术语表（中英对照）
- [ ] 核实价格、功能等时效性信息（标注日期）

### 输出文件
- [ ] 保存到 `scripts/[视频名]/visual-reference.md`
- [ ] 保存到 `scripts/[视频名]/tech-verification.md`
- [ ] 更新本资源库（`docs/05-素材资源库.md`）
