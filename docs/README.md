## 快速开始

### 第一步：确定方向
阅读 [01-项目总览](./01-项目总览.md)，选择你的主要方向。

### 第二步：选择主题
阅读 [02-选题方向](./02-选题方向.md)，选择第一个视频的主题。

### 第三步：撰写脚本
使用 [06-脚本大纲模板](./06-脚本大纲模板.md)，填写具体内容。

### 第四步：准备素材
参考 [05-素材资源库](./05-素材资源库.md)，收集所需素材。

### 第五步：录音处理
参考 [03-录音与声纹指南](./03-录音与声纹指南.md)，完成录音。

### 第六步：视频制作
参考 [04-视频制作流程](./04-视频制作流程.md)，完成视频制作。
