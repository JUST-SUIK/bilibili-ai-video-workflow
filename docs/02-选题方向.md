# AI知识分享视频 - 选题方向详解

## 选题原则

1. **你熟悉或正在学习的** - 真实体验更有说服力
2. **有明确受众的** - 知道谁会看
3. **可执行的** - 能用现有资源制作
4. **有差异化的** - 和别人做的不一样

## 五、推荐起号选题（前5个视频）

### 第1个视频：自我介绍 + 方向预告

**标题**：`计算机专业学AI：我为什么开始做这个频道`

**内容**：
- 个人背景介绍
- 为什么想做AI分享
- 频道内容规划
- 对观众的期待

**时长**：3-5分钟

**目的**：建立人设，让观众认识你

---

### 第2个视频：AI工具入门

**标题**：`AI工具入门：ChatGPT/Claude/文心一言怎么选？`

- 各自的特点和优势
- 适用场景对比
- 个人使用建议

**时长**：8-10分钟

**目的**：实用内容，吸引目标受众

## 六、选题选择建议

### 如果你想快速起号：
选择 **系列1（AI概念轻松懂）** 或 **系列2（AI工具入门指南）**
- 受众广泛
- 制作相对简单
- 容易获得搜索流量

### 如果你想建立专业形象：
选择 **系列3（AI实战案例）** 或 **系列4（AI工作流搭建）**
- 展示技术深度
- 实用价值高
- 容易获得收藏

### 如果你想突出个人特色：
选择 **系列6（AI创意工坊）** 结合 **系列5（AI深度思考）**
- 差异化明显
- 展示个人风格
- 容易形成品牌

---

## 七、选题注意事项

### ✅ 好选题的特点
- 有明确的受众和价值
- 你能讲出独特见解
- 有足够的素材支撑
- 制作难度可接受

### ❌ 避免的选题
- 太宽泛没有重点
- 和你的专业背景不符
- 缺乏实际案例
- 制作周期太长
